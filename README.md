## Running

```
npm install
npm start
```

## Scaffolding

Feathers has a powerful generator. Here's just a few things that it can do:

- **Generate a new Service:** `yo feathers:service`
- **Generate a new Hook:** `yo feathers:hook`
- **Generate a new Model:** `yo feathers:model`
